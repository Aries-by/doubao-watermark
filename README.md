# 豆包去水印 (Doubao Watermark Remover)

一个轻量级的用户脚本，完美兼容 **Tampermonkey (油猴)** 和 **ScriptCat (脚本猫)**，专为豆包 (doubao.com) 网页版设计。通过拦截底层数据接口，自动提取并替换无水印的原图链接，让你在网页端直接预览和保存纯净的 AI 绘图作品。

## ✨ 核心特性

* **🚀 无损原图提取**
  深入拦截 `JSON.parse` 解析过程，在前端渲染前，自动将带有水印的预览图、缩略图强制替换为底层的无水印原图 (`image_ori_raw`)。
* **🎛️ 极简可视化面板**
  告别繁琐的 F12 开发者工具。脚本自带一个精致的悬浮球（带有“豆”字），点击即可展开日志面板，实时查看成功拦截的图片数量和状态。
* **🖱️ 自由拖拽与位置记忆**
  悬浮球支持在屏幕内自由拖拽。通过 `localStorage` 自动记忆你的摆放习惯，刷新页面位置不丢失，绝不遮挡网页主体内容。
* **⚡ 纯净高效**
  零外部依赖，不额外发送任何网络请求，原生级别的轻量化运行。

## 📦 安装指南

1. **安装脚本管理器**
   首先，确保你的浏览器已经安装了以下任意一款扩展程序：
   * [ScriptCat (脚本猫)](https://docs.scriptcat.org/) - 推荐，国内访问更稳定，功能强大。
   * [Tampermonkey (油猴)](https://www.tampermonkey.net/) - 经典的老牌脚本管理器。

2. **添加本脚本**
   * **一键安装**：在 GitHub 仓库中找到 `豆包去水印.js` 文件，点击 `Raw` 按钮，脚本猫或油猴会自动弹出安装提示，点击“安装”即可。
   * **手动安装**：打开脚本猫或油猴的管理面板，点击“添加新脚本”，将本项目中的 `豆包去水印.js` 源码完整复制并粘贴进去，按 `Ctrl + S` (或 `Cmd + S`) 保存。

## 💡 使用说明

1. 正常访问 [豆包 (doubao.com)](https://www.doubao.com/) 并登录你的账号。
2. 在对话框中输入你的绘画提示词，让豆包生成图片。
3. 页面右下角会出现一个绿色的“豆”字悬浮球。
4. 当图片生成完毕或你在浏览历史作品时，脚本会自动在后台完成去水印工作。你可以点击悬浮球打开面板，查看类似 `[拦截 X 张图] [已换为无水印原图]` 的成功日志。
5. 此时页面上展示的已经是无水印原图，直接右键另存为即可。

## 🛠️ 技术实现浅析

本脚本的核心原理是**网络请求的数据劫持**。由于豆包后端的原始返回数据中其实已经包含了无水印的底层图片链接（`image_ori_raw.url`），脚本通过重写原生的 `JSON.parse` 方法，在业务代码读取数据前，将展示用的带水印链接全部替换为原图链接，从而实现了“欺骗”前端渲染的去水印效果。
